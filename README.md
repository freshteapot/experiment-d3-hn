experiment-d3-hn
================

An experiment to explore using d3.js to visualise data collected from Hacker News.

#The Plan

* Gather data using the php script via a cronjob.
* Process the log of data and present it in a visual manner.

#Status

- First attempt at a bar chart.
- 2 weeks of data collected.


##What is d3.js

[d3.js](http://d3js.org/ "Data-Driven Documents") an amazing javascript library for visualising STUFF.

##What is Hacker News

[Hacker News](http://news.ycombinator.com/ "Hacker News") a website to get information mostly related to tech.
